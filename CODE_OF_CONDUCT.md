# Code of Conduct

## Our Pledge

We pledge to make participation in NaviProxy a respectful and constructive experience for everyone.

## Expected Behavior

- Be respectful and patient.
- Assume good intent.
- Give clear, actionable feedback.
- Keep discussions focused on the project.
- Respect maintainers' time and project scope.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publicly posting private information.
- Spamming issues, discussions, or pull requests.
- Demanding unpaid work from maintainers.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block users who violate this code of conduct.

## Scope

This code applies to the repository, issues, pull requests, discussions, release channels, and any official NaviProxy community spaces.
